# QGrapho assets

Visual assets for documentation and social sharing.

## README diagrams

The main [README](../README.md) uses **Mermaid** diagrams — they render natively on GitHub without broken image links.

## SVG files (docs and exports)

| File | Use |
|------|-----|
| `qgrapho-logo.svg` | Logo mark |
| `qgrapho-overview.svg` | Platform overview (docs) |
| `qgrapho-modalities.svg` | Modality router (docs) |

Reference in markdown with a **raw GitHub URL** when embedding as images:

```markdown
![Overview](https://github.com/quanvio/qgrapho/raw/main/docs/assets/qgrapho-overview.svg)
```
